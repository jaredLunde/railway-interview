# Deployer

## Local development

### Prerequisites

- [**Bun**](https://bun.sh/)
  ```sh
  curl -fsSL https://bun.sh/install | bash
  ```

### Setup

1. Clone the repository

```sh
git clone https://github.com/jaredLunde/railway-interview
```

2. Install dependencies

```sh
bun install
```

3. Start the development environment

```sh
bun dev
```
